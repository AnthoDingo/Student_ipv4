# IPv4

Programme d'apprentissage des classes IPv4.
Outils à but non lucratif.

## Requis
[.NET 7 Desktop Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/7.0)

## Mode Devoir
Lancer le programme en Mode Administrateur pour changer les paramètres de l'application.
Les paramètres sont stockés dans le fichier "setting.dat" dans le répertoire du programme.
